# 🛒 Ecommerce Sales Analysis Project

## 📌 Overview
This project analyzes ecommerce sales data to uncover insights on sales, profit, and customer trends.

## 📂 Project Structure
- data/ → Dataset
- notebook/ → Python analysis
- images/ → Dashboard screenshots

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI

## 📊 Key Insights
- West region has highest sales
- Technology category gives highest profit
- Seasonal trends observed in monthly sales
